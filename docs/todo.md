# OCGP 專案待辦與里程碑

## 里程碑 1：基礎安全與持久化
- [x] SQLite 持久化（帳號/Session/房間/玩家清單）；`OCGP_DB_PATH` 可覆寫。  
- [x] 密碼加鹽雜湊（隨機 salt + SHA-256）。仍待 TLS 與 DB 權限控管。  
- [x] 基礎日誌（登入/登出/房間/棋步）輸出 `backend/out/logs/ocgp.log`。  
- [x] 備份與還原指引（`docs/backup.md`）。  

## 里程碑 2：即時溝通與房間功能
- [x] WebSocket 推播（房間狀態、聊天），失敗時回落輪詢。  
- [x] 房間設定編輯（房主可改名稱/遊戲/私密與邀請碼），搜尋支援名稱/遊戲/邀請碼。  
- [x] 私人房邀請碼加入，加入/離開/開局/重開/聊天系統訊息推送。  

## 里程碑 3：遊戲體驗與效能
- [ ] 倒數計時與回合時間（符合象棋/五子棋 1–3 秒回應要求）。  
- [x] 對局斷線保留 30 秒可重連：離線標記+超時重置，重連保留棋局。  
- [x] Ping 偵測（前端按鈕 `/api/ping` 回報延遲）。  
- [x] 重新開始對局流程優化（直接重新開局、保留玩家順序）。  

## 里程碑 4：品質與部署
- [ ] 自動化測試（後端 API、棋規、WebSocket 流程）。  
- [ ] Docker / Compose 部署設定（環境變數、健康檢查）。  
- [ ] 監控與告警（基礎指標、錯誤率）。
